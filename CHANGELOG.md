## [1.0.0] - 2025-11-20

### Added
- Initial release of BatchCore SDK
- Core client (`BatchCoreClient`) for batch processing operations
- Configuration options (`BatchCoreOptions`) for SDK customization
- Support for submitting batch requests
- Support for checking batch status
- Support for cancelling batch operations
- Dependency injection extensions via `AddBatchCore()`
- Comprehensive XML documentation
- Unit tests with xUnit
- Sample application demonstrating SDK usage
- NuGet package configuration with Source Link support
